# gambit-cli-hello-world

## Description
A POC for gambit programming language.

A cross language compilar/interpreture for
java, javascript, python, ruby, and scheme

## Tech stack
- gambit
- java
- javascript
- python
- ruby
- scheme

## Docker stack
- ubuntu:22.04

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
